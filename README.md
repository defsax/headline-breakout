# Headline Breakout

Basically a gamified RSS reader. Works on web and mobile browsers!

Uses Fetch API and parses XML from a few different news outlets/aggregators including:
The New York Times<br>
CBC<br>
BBC<br>
HuffPo<br>
HackerNews<br>
